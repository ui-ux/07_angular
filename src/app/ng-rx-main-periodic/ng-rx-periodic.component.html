State

State structure using interfaces

Actions

  • createAction: Function to create a new action.
  • props: Function to define action properties.
  • PeriodicElement: An interface that describes the data model of an element.

Reducer

A reducer gets the current state and an action, and then returns a new state based on that action.
  • createReducer: Function to create a reducer.
  • on: A function for processing specific actions in the reducer.
  • PeriodicElementState: An interface that describes the state model for elements.
  • loadElements, setFilter: Import actions from the file actions.

Selectors

Selectors defines functions used to sample and compute data from the global state of the NgRx Store.
Selectors allow components to subscribe to specific state changes and receive only the data they need.
  • createSelector: Function for creating selectors.
  • AppState, PeriodicElementState: Interfaces that describe the structure of the application state and the state of periodic elements.
  • selectPeriodicElementState: A selector function that receives the entire state of the application (AppState) and returns only the state of periodic elements (PeriodicElementState).
  • selectFilteredElements: A selector that uses selectPeriodicElementState to retrieve the state of periodic elements.
  • selectFilter: A selector that uses selectPeriodicElementState to get the state of periodic elements and returns only the filter value.
